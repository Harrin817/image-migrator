# image-migrator

# 🐱 薄荷猫の图床全自动迁移工具（终极优化版）

一个基于 **Python + Streamlit** 构建的智能图床迁移工具，支持从文本、代码、Markdown、CSS 以及 Word 文档中自动提取图片链接，并通过 Postimages 进行批量转存，实现**无损替换与一键迁移**。

> ✨ 全程本地运行 | 🚀 全自动流水线 | 🧠 AI 智能提取 | 📦 自动打包导出

---

## ✨ 核心功能

### 🚀 全自动流水线
从 **链接提取 → 图床转存 → URL 替换 → 打包导出** 全流程自动完成，无需手动干预。

### 🧠 AI 智能识别
结合大语言模型（LLM），可从复杂文本中自动识别图片 URL（支持 Markdown / HTML / 混合文本）。

### 📄 Word 深度支持（.docx）
直接解析 Word 底层 XML 结构，实现：
- 图片直链精准提取
- 不破坏排版结构的无损替换

### ⚡ 内存级处理
所有操作均在内存中完成：
- 无临时文件残留
- 更高安全性
- 更快执行速度

### 🔁 直链优化提取
自动从 Postimages 页面中解析：
- `direct link`
- `og:image`
实现真正可用的 CDN 直链

### 📦 批量处理 & 自动打包
- 支持多文件同时处理
- 自动生成 ZIP 下载包
- 自动生成 URL 映射对照表

### 💾 配置持久化
- 自动保存 API Endpoint & API Key
- 下次启动自动加载

---

## 🛠 技术栈

| 模块 | 技术 |
|------|------|
| 前端界面 | Streamlit |
| AI 接口 | OpenAI SDK |
| 网络请求 | curl_cffi（模拟浏览器绕过防护） |
| Word 解析 | zipfile + XML 解析 |
| 数据处理 | Python 标准库 |
